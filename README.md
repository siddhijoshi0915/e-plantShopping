# e-plantShopping

A React and Redux-based e-commerce web application for buying plants online.  
This project allows users to browse different categories of plants, add them to a shopping cart, update quantities, and manage cart items dynamically using Redux.

## Features
- Browse multiple plant categories
- Add plants to cart
- Increase/decrease quantity
- Remove items from cart
- Real-time cart total calculation
- Redux state management

## Tech Stack
- React JS
- Redux Toolkit
- HTML/CSS
- JavaScript

## Project Goal
To build a functional shopping cart system using React and Redux for state management.
